BlueTrace Lab

Detection Lab

A home detection lab built to practice log analysis, alert triage, and incident response — using real tools in a containerised, local environment.

What it demonstrates
  • Structured log review, alert triage, and baseline detection for common attacker patterns
  • SIEM concepts with Splunk for search, dashboards, and detection rule development
  • IDS tuning with Suricata and MITRE ATT&CK mapping
